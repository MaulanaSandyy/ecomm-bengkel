AUTOGARAGE - SISTEM MANAJEMEN BENGKEL & SPAREPART
==================================================

DESKRIPSI PROYEK
AutoGarage adalah aplikasi web untuk mendigitalisasi manajemen bengkel otomotif. Sistem ini melayani dua sisi: Administrator (untuk mengelola operasional) dan Customer (untuk booking service dan beli sparepart).

FITUR UTAMA
- Admin: Mengelola katalog sparepart, harga jasa, antrean booking, konfirmasi pembayaran, dan cetak nota transaksi.
- Customer: Melihat katalog produk, memasukkan ke keranjang, checkout pembayaran (termasuk via QRIS), dan memantau riwayat service.

TEKNOLOGI YANG DIGUNAKAN
- PHP Native
- MySQL
- HTML, CSS, JavaScript
- Bootstrap 5

PANDUAN INSTALASI LOKAL
1. Pindahkan folder proyek ini ke dalam direktori server lokal Anda:
   - XAMPP: C:/xampp/htdocs/nameproject
   - Laragon: C:/laragon/www/nameproject
2. Buka browser dan jalankan phpMyAdmin (http://localhost/phpmyadmin).
3. Buat database baru dengan nama: db_bengkel
4. Import file 'db_bengkel.sql' ke dalam database tersebut.
5. (Opsional) Sesuaikan detail koneksi database di file 'includes/koneksi.php' jika password MySQL Anda bukan kosong.
6. Buka aplikasi di browser melalui URL: http://localhost/autogarage
